Digital Capability Lookup (DCL) interface specification

API Endpoint
https://dbc-dcl.testpoint.io/api/ausdigital/v0

The “Digital Capability Lookup” (DCL) specification defines DNS based method to discover which DCP(SMP) is the authoritative source for a given business. AusDigital version.
